# nema.one

React-Grundgerüst für die Website `nema.one`.

## Entwicklung starten

```bash
npm install
npm run dev
```

## Build erstellen

```bash
npm run build
```

## Projektstruktur

```text
nema.one/
├── public/
├── src/
│   ├── App.jsx
│   ├── main.jsx
│   └── styles.css
├── index.html
├── package.json
└── vite.config.js
```
